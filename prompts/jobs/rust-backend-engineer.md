# Rust 后端工程师

## 岗位职责

- 负责后端服务的设计、开发和维护
- 参与系统架构设计，保证系统的高可用性和高性能
- 编写高质量、可维护的代码
- 参与代码审查和技术分享

## 任职要求

- 3年以上后端开发经验
- 精通 Rust 语言，熟悉异步编程（Tokio）
- 熟悉分布式系统设计，了解微服务架构
- 熟悉数据库（PostgreSQL/MySQL）和缓存（Redis）
- 有高并发系统开发经验者优先
- 良好的代码规范和文档编写习惯

## 加分项

- 有 Rust 开源项目贡献经验
- 熟悉云原生技术栈（Kubernetes、Docker）
- 有技术博客或分享经验
- 了解 WebAssembly 或嵌入式开发
