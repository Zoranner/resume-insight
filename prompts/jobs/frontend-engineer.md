# 前端工程师

## 岗位职责

- 负责 Web 前端页面的开发和维护
- 与设计师、产品经理协作完成产品需求
- 优化前端性能，提升用户体验
- 参与前端技术选型和架构设计

## 任职要求

- 3年以上前端开发经验
- 精通 Vue.js 或 React.js 框架
- 熟悉 TypeScript、ES6+ 语法
- 熟悉前端工程化（Webpack/Vite）
- 了解 Web 性能优化和跨浏览器兼容
- 有移动端 H5 开发经验者优先

## 加分项

- 有组件库开发经验
- 熟悉 Nuxt.js/Next.js 等 SSR 框架
- 了解前端可视化（D3.js/ECharts）
- 有开源项目或技术博客
