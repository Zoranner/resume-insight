# DevOps 工程师

## 岗位职责

- 负责 CI/CD 流程的搭建和优化
- 管理和维护云基础设施（AWS/阿里云）
- 监控系统运行状态，及时处理故障
- 推动自动化运维和效率提升

## 任职要求

- 3年以上 DevOps 经验
- 熟悉 Kubernetes、Docker 等容器技术
- 熟悉 CI/CD 工具（Jenkins/GitLab CI）
- 熟悉 Linux 系统管理和 Shell 脚本
- 了解监控系统（Prometheus/Grafana）
- 有云平台运维经验者优先

## 加分项

- 有大规模集群管理经验
- 熟悉 Infrastructure as Code（Terraform/Ansible）
- 了解安全加固和合规要求
- 有 SRE 实践经验
